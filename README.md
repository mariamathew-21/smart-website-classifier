# Smart Website Classifier

## Project Overview
Developed a web-based machine learning application that classifies website reviews using natural language processing and machine learning techniques.

## Features
- Text classification
- FastText embeddings
- Logistic regression model
- Web application interface
- Review classification predictions

## Technologies Used
- Python
- Flask
- FastText
- Logistic Regression
- Machine Learning
- NLP

## Project Workflow
- Preprocessed review text data
- Built language embeddings using FastText
- Trained classification model
- Integrated model into a web application
- Generated predictions through user input

## Key Skills Demonstrated
- Machine learning
- Natural language processing
- Model deployment
- Python development
- Web application integration
